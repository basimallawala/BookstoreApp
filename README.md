This is the final project for the COE528 Java course written by Basim Allawala, Ishan Dubey and Sullay Sinnah
in the W2026 semester. It implements a simple Java bookstore using the Swing GUI package a state design pattern.

Feature tracker:
- Multiple screens implemented in Java Swing
  - [x] Login screen
  - [x] Owner start screen
  - [ ] Owner books screen
  - [ ] Owner customers screen
  - [ ] Customer start screen
  - [ ] Customer buy screen
- [ ] Edge case testing
- [x] Loading and saving data to persistent memory
